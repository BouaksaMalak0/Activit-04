Chapitre 4 - Le World Wide Web

Introduction

L'introduction de ce chapitre met en évidence la différence entre Internet et le World Wide Web (WWW). Le WWW est décrit comme un système hypertexte public qui permet aux utilisateurs de consulter des pages web accessibles sur des sites internet. Les principes du WWW sont expliqués, y compris l'utilisation de liens hypertexte pour connecter des documents textuels, ainsi que le rôle des URL pour localiser les ressources en ligne.

Aspects techniques du Web

Cette partie du chapitre décrit les trois mécanismes de base du Web : les URL, qui permettent l'accès à un ensemble de ressources, le protocole HTTP, qui permet la communication client-serveur, et HTML, qui est le langage de description des pages web. Les caractéristiques de chaque mécanisme sont également expliquées, telles que le protocole sans état du HTTP.

Page web et site web

Cette section définit une page web comme un document multimédia stocké dans un fichier HTML ou XHTML, qui peut contenir du texte, des images et des liens hypertexte. Elle explique également la différence entre une page web et un site web, qui est un ensemble de pages web liées entre elles par des liens hypertextes, organisées autour d'une thématique commune et émanant d'une même entité.

Création d'une page web HTML

Cette partie fournit des informations techniques sur la création d'une page web HTML, y compris la structure de base du langage HTML, les balises nécessaires pour délimiter le début et la fin du document, et les balises pour les titres et les paragraphes. Elle explique également que les fichiers HTML peuvent être créés dans un éditeur de pages web ou dans un éditeur de texte.
